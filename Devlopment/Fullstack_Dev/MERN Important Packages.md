
# Node Packages

## Backend

Create and Navigate to new Directory and Setup
```sh
mkdir backend
cd mern-backend
npm init -y
```

Install Packages from `Package.json`
```
npm i
```

- express
- mongoose
- dotenv
- cors
- body-parser
- nodemon
- axios

## Frontend

Navigate to Project root and Create react App
```
cd..
npx create-react-app client
cd client
```

Install Packages from Package.json
```
npm i
```

- react
- react-router-dom
- axios
- react-hot-toast
- redux
- react-redux
